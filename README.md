# parenthood

A partner IAT for the "Transitioning and Adjusting to Parenthood" study.

## Usage

### Dependencies

- [Leiningen](http://leiningen.org/)
- [Heroku Local](https://devcenter.heroku.com/articles/heroku-local)

### Local Development

    heroku local

### Deploy to Heroku

    git push heroku dev-branch:master
